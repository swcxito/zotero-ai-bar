## 使用教程

以下为快速教程，详细教程请访问[官网](https://zotero.fukeke.com/zh-cn/)：

1. 安装插件
2. 打开模型设置
3. 添加供应商，填写 API Key 和 model
4. 关闭模型设置页，设置会自动保存
5. 开始使用吧！
